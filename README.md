HOW TO GET CASSANDRA
====

if you have no python: 
	
		brew install python

then:

		brew install cassandra


then: 

		pip install csql



HOW TO RUN CASSANDRA
====


		cassandra -f


HOW TO RUN CSQL TO WORK IN CASSANDRA
====

	
		cqlsh


brings up CQL command line. Try it: 


		cqlsh > HELP
